# Healthcare-Analytics-Dashboard-
Built a Power BI dashboard to uncover insights into patient care efficiency, satisfaction, and healthcare service demand.  

Objective:
The objective of this project was to develop an interactive Healthcare Analytics Dashboard that provides insights into patient care efficiency, service demand, and satisfaction levels. The dashboard enables healthcare administrators to identify operational bottlenecks, improve patient experience, and make data-driven decisions to optimise service delivery.
